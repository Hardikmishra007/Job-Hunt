# 💼 Job Hunt

An online platform designed to connect job seekers with employers by providing an easy way to post jobs, apply for opportunities, and manage applications. Built with modern web technologies, this project ensures a smooth and interactive recruitment experience.

---

# 🚀 Features

User-friendly job seeker and recruiter registration/login

Job posting and job application management

Candidate profile creation with resumes and skills

Employer dashboards for tracking applications

Secure authentication and session handling

Responsive UI for desktop and mobile devices

---

# 🛠️ Tech Stack

Frontend: React.js / HTML / CSS / JavaScript

Backend: Node.js / Express.js

Database: MongoDB

Authentication: JWT (JSON Web Tokens)
